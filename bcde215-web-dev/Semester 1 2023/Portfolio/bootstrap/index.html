Welcome to a Django Setup Tutorial


This website is a tutorial that provides step-by-step instructions for setting up a Django project. Whether you are new to Django or looking to enhance your skills, this tutorial will guide you through the process of setting up the necessary AWS infrastructure, configuring instance settings, installing a web server, and setting up your website. Follow along to learn how to create a Django website from scratch and explore topics such as backup setup and making a Django website using this comprehensive tutorial.

Note: There will most likely be errors along the way as to make this site, even if you follow this process to a T.
You can do the following:

  • sudo rm -r [directory] - this one is the most radical as it's just restarting completely
  • Look at the Apache Logs: /var/log/apache2/
  • Try and debug with whatever the PuTTY gives you - some of these can be straightforward, others not so much
  • Ask people around you then ask the Lecture/s - Apply the "See Three Before Me" rule
