Amazon EC2 Web Server Installation Instructions

3 April 2023, Chris Bartlett

Read each section of instructions carefully before starting!

Indented Lines

Indented lines of text below that use the Courier New font are lines of code that you could copy and paste into the terminal session when connected to your instance. To paste copied code, right click on the PuTTY terminal window. However, please do not simply copy-paste-copy-paste… - you need to understand what you are doing, so take your time and read the instructions carefully.

Update Linux packages

Make sure your instance has an up-to-date list of program packages that are available for installation:

sudo apt update

Install the Apache webserver program

Install Apache using apt:

sudo apt install -y apache2

Get the public IP address of your Amazon EC2 server instance (in the EC2 console) and go to it with your web browser. The “Apache2 Ubuntu Default Page” should show in your browser. If it does, you have set up a web server. Well, how easy was that?
